# Modern Tech Blog – Bootstrap 5

A responsive and attractive blog webpage built using **Bootstrap 5**, featuring a navigation bar, hero section, blog cards, dark mode toggle, and footer with social icons. This project demonstrates the use of Bootstrap components for rapid UI development and responsive layout structure.

---

##  Features

- Responsive Navbar with brand and menu
- Modern hero section with background image
- Blog post cards (image + title + description + button)
- Hover effects on cards
- Dark Mode / Light Mode toggle
- Fully responsive grid layout
- Professional UI design using Bootstrap utility classes
- Footer with social media icons

---

## Technologies Used

HTML5 – Used to structure the webpage.

CSS3 – Used for additional custom styling and appearance.

Bootstrap 5 (CDN) – Used for responsive layout and UI components.

FontAwesome – Used for icons in navbar and footer.

Unsplash – Used for high-quality blog post images.

JavaScript – Used to implement the theme switch (light/dark mode).
